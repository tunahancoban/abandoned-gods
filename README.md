# Abandoned Gods - 2D Platformer

Abandoned Gods, Unity motoru ve C# dili kullanılarak geliştirilmiş, mekanik odaklı bir 2D platform oyunudur. Bu depo, projenin temel oyun mantığını, mimarisini ve sistem tasarımlarını içermektedir.
Projede kullanılan görsel assetler, lisans kısıtlamaları ve repo boyutunu optimize etmek amacıyla bu depoya dahil edilmemiştir. Oyunun tam görselliği için yukarıdaki Itch.io linkini ziyaret edebilirsiniz.
